# Tasting Journal

A personal tasting note app for recording wines, sake, and other fermented beverages.

## Features

- Create tasting notes
- Browse tasting history
- View tasting details
- Rate drinks
- Add aroma and flavor notes

## Future Features

- Image upload
- Tag management
- Search & Filter
- AI tasting assistant
- Tasting map
- Statistics

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- localStorage (MVP)
- Supabase (Future)
